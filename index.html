• pwd:返回当前目录地址
  • cd:切换目录；cd .. 返回上级
  • ls:查看目录下的文件； ls 文件夹名==查看指定文件夹内容； ls -a==查看目录下全部文件; ls -l==查看全部文件的权限
  • mkdir 文件夹名;创建目录
  • touch 文件名；创建文件；touch 路径/文件名
  • cat 文件名;查看文件内容
  • rm 文件名;删除文件 rm -r 文件夹/ ==删除路径下所有文件
  • rmdir 文件夹名;只能删除空文件夹
  • mv 文件名 文件夹 ==移动；mv 路径/文件名 文件夹/重命名文件名
  • cp 拷贝和mv用法一样
  • head -5 文件名==查看文件的前五行
  • tail -5 文件名==查看文件后五行
  • history 查看当前操作命令历史
  • > 和 >>重定向 >替换式 >>追加式
  • grep 正则
  • clear 清屏
  • esc :qw保存推出编辑窗口
        首先：初始首次的用户名和邮箱后
        打开一个文件夹，实用git init 设置成我们的仓库
        然后用代码编辑器新建一个文件。
        第一步，用命令git add告诉Git，把文件添加到仓库：
        git add .
        第二步，用命令git commit告诉Git，把文件提交到仓库：
        $ git commit -m "一句描述"
        git status 查看状态如果有更新的，会显示红色的名字
        查看被更新的文件哪些东西更新了呢？
        git diff
        当我们修改了那么多次后，如何看提交的几个版本：
        git log  --pretty=oneline
        回退到以前版本或回到最后一次提交都使用这个命令
        git reset --hard commit_id
        git reste --hard HEAD //回退到上一个版本
        如果第二天忘了id怎么办，查看git log也不行，还有办法
        git reflog
